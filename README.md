# 🍽️ Menú Semanal

Planificador quincenal de comidas y compras para Alba y Sergi.

## Objetivo

Reducir la carga mental asociada a la planificación de comidas, optimizar la compra semanal y mantener una alimentación equilibrada y realista.

## Funcionalidades

### 📅 Planificación quincenal

* Menú de 14 días
* Comidas y cenas
* Organización visual por semanas

### 🛒 Lista de la compra

Organizada por secciones de Mercadona:

* Fruta y verdura
* Carnicería
* Pescadería
* Lácteos
* Conservas
* Pasta y arroz
* Panadería
* Agua
* Otros

### ❤️ Favoritos

Sistema de valoración de recetas para reutilizar los platos mejor puntuados.

### ⚡ Airfryer

Identificación rápida de recetas compatibles con Airfryer.

### ❄️ Congelador

Recordatorios sobre qué productos sacar del congelador.

### 🥣 Batch Cooking

Preparaciones recomendadas para adelantar trabajo los domingos.

### 💧 Consumo de agua

Control quincenal:

* 5 litros diarios
* 70 litros por quincena
* 14 garrafas de 5 litros

## Preferencias

### Alba

* Cocina mediterránea
* Verduras variadas
* Mozzarella
* Burrata
* Quesos suaves

### Sergi

* Todo tipo de quesos

### Ambos

* Compras principalmente en Mercadona
* Uso frecuente de Airfryer
* Una comida especial semanal:

  * Pizza
  * Sushi
  * Hamburguesa
  * Mexicano
  * Asiático

### Evitar

* Ensaladilla rusa

## Tecnología

* Progressive Web App (PWA)
* HTML
* CSS
* JavaScript
* LocalStorage
* Service Worker
* Funciona offline
* Instalable en iPhone

## Actualización

La aplicación está diseñada para que únicamente sea necesario actualizar el archivo de datos cada 15 días, manteniendo intacta toda la experiencia de usuario.
